# Real Estate Website

## Requirements
- Node.js (preferably v22.11.0)
- npm

## Installation
1. **Clone the repository:**
   ```bash
    git clone https://github.com/P1ayMouse/real-estate
    cd real-estate
   ````
2. **Install dependencies:**
   ```bash
    npm install
   ```

3. **Run scripts:**
   
   * Run SASS (compile SCSS to CSS):
     ```bash
      npm run sass
     ```
   * For code linting with ESLint and Stylelint:
     ```bash
      npm run lint
     ``` 

   * Launch the project:
     ```bash
      npm run start
     ```
